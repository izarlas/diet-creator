# Diet Creator

This is a React TypeScript and NestJs application that aims to enable creating diet plans smoothly

## Running the application

- To run the client navigate to `/client` and run `npm run start` and access the application locally at `http://localhost:5173/`
- To run the server navigate to `/server` and run `npm run start` and access the application locally at `http://localhost:3000/`
- Run the entire application using compose, with `docker-compose up`
